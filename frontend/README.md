# Cerevyn frontend

React + TypeScript + Vite + Tailwind. See the **[repository root README](../README.md)** for setup, environment variables, and how to run the full stack.

Development:

```bash
npm install
npm run dev
```

Optional: copy `.env.example` to `.env.local` and set `VITE_API_BASE_URL` if the API is not on `http://127.0.0.1:8000`.
